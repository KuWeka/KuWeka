## 🚀 Featured Project

### 🖥️ HelpdeskPolda — Sistem IT Helpdesk Polda Kalsel

[![Repo](https://img.shields.io/badge/GitHub-projectpolda-6A0DAD?style=for-the-badge&logo=github)](https://github.com/KuWeka/projectpolda)
![Stack](https://img.shields.io/badge/Stack-React%20·%20Node.js%20·%20MySQL%20·%20Redis%20·%20Docker-20232A?style=for-the-badge)

> Sistem helpdesk IT berbasis monorepo untuk Polda Kalimantan Selatan, mencakup manajemen tiket, assignment teknisi, dan komunikasi real-time.

| Fitur | Deskripsi |
|-------|-----------|
| 🎫 Ticket System | Buat, assign, dan update tiket dengan dashboard monitoring |
| 💬 Real-time Chat | Komunikasi langsung antar pengguna via Socket.IO |
| 📊 Observability | Logging terpusat (Winston) + Metrics (Prometheus & Grafana) |
| 🔐 Auth & Security | JWT-based authentication & role-based authorization |
| 🐳 Containerized | Deployment siap pakai dengan Docker |
