### 🛡️ SIGAP — Sistem Informasi Gangguan dan Aduan Polri
> **IT Helpdesk internal production-grade untuk Polda Kalimantan Selatan** &nbsp;⭐ *Main Project*

<div align="center">

[![Repo](https://img.shields.io/badge/GitHub-Sigap--polda--kalsel-7C3AED?style=for-the-badge&logo=github&logoColor=white)](https://github.com/KuWeka/Sigap-polda-kalsel)
![Stack](https://img.shields.io/badge/Stack-Next.js_·_Express_·_MySQL_·_Socket.io-6D28D9?style=for-the-badge)
![Deploy](https://img.shields.io/badge/Deploy-Vercel_+_Railway_+_Cloudinary-5B21B6?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Active_·_Production_Ready-22C55E?style=for-the-badge)

</div>

Platform helpdesk IT internal yang menggantikan pencatatan manual gangguan TI di lingkungan Polda Kalsel. Dibangun sebagai **TypeScript monorepo** (frontend + backend + shared package) dengan arsitektur production-grade — real-time, role-based, fully auditable.

**✨ Fitur Lengkap:**

| Modul | Deskripsi |
|:---|:---|
| 🎫 **Ticket Lifecycle** | Alur end-to-end: PENDING → PROSES → SELESAI / DITOLAK / DIBATALKAN |
| 👥 **RBAC 4 Role** | Satker, Bidtekkom, Padal, Teknisi — hak akses terpisah per role |
| 🔔 **Real-Time Notifikasi** | Socket.io per-user room, event dikirim dalam < 5 detik |
| ⭐ **Rating Gate** | Satker wajib beri rating sebelum bisa buat tiket baru |
| 📊 **Laporan Bulanan** | Export PDF (PDFKit) & Excel (ExcelJS) dengan ringkasan statistik |
| 🔍 **Audit Log** | 15 tipe event tercatat lengkap — tidak bisa dihapus |
| 🔒 **Enterprise Security** | JWT + bcrypt + Helmet + rate limiting + optimistic locking race condition |
| 🏷️ **Nomor Tiket Atomik** | Format `TKT-2026-00001` via DB transaction, aman dari concurrent request |
| 📁 **File Upload** | Langsung ke Cloudinary CDN, max 5MB, 10 file per tiket |
| 📈 **Dashboard per Role** | Statistik, bar chart, line chart, quick-assign untuk Bidtekkom |
| 🧪 **117 Tests Pass** | Property-based testing dengan `fast-check` — semua invariant diverifikasi |
| ⚡ **Auth Cache** | In-memory TTL 60s mengurangi DB query per request |

**Tech Stack:**
`Next.js 14` `React 18` `TypeScript` `Express` `Prisma ORM` `MySQL 8` `Socket.io` `shadcn/ui` `TanStack Query v5` `TanStack Table` `Recharts` `React Hook Form` `Zod` `PDFKit` `ExcelJS` `JWT` `bcryptjs` `Helmet` `Cloudinary` `Vercel` `Railway`
