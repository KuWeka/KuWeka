## 🚀 Featured Projects

### 🛡️ HelpdeskPolda — Sistem IT Helpdesk Polda Kalsel ⭐ *Main Project*

<div align="center">

[![GitHub Repo](https://img.shields.io/badge/GitHub-Helpdesk--IT--POLDA-181717?style=for-the-badge&logo=github)](https://github.com/KuWeka/Helpdesk-IT-POLDA)
[![Version](https://img.shields.io/badge/version-1.0.0-9B30FF?style=for-the-badge)](#)
[![Stack](https://img.shields.io/badge/Stack-React_·_Node.js_·_MySQL_·_Redis_·_Docker-6D28D9?style=for-the-badge)](#)
[![CI/CD](https://img.shields.io/badge/CI%2FCD-GitHub_Actions-2088FF?style=for-the-badge&logo=githubactions&logoColor=white)](#)
[![Status](https://img.shields.io/badge/Status-Active_·_Production_Ready-22C55E?style=for-the-badge)](#)

</div>

> 🏗️ Platform helpdesk IT berbasis **monorepo** untuk Polda Kalimantan Selatan — production-grade dengan real-time chat, multi-role dashboard, observability penuh, dan CI/CD pipeline siap pakai.

**✨ Fitur Unggulan:**

- 🎫 **Manajemen Tiket** — Buat, assign, tracking, dan riwayat tiket IT secara penuh dengan dashboard monitoring
- 💬 **Real-time Chat** — Komunikasi langsung antara user dan teknisi via Socket.IO
- 👥 **Multi-role Panel** — Dashboard terpisah untuk User, Teknisi, dan Admin
- 📊 **Dashboard & Insight** — Visualisasi statistik tiket, aktivitas, dan performa teknisi via Recharts
- 🔔 **Notifikasi Real-time** — Toast notification dan update status instan
- 🔐 **Enterprise Security** — JWT + Cookie auth, CSRF protection, rate limiting, Helmet.js
- 📈 **Observability** — Prometheus metrics, Winston logging, Grafana dashboard, health check endpoint
- 🐳 **Docker Ready** — Build & deploy siap container dengan staging scripts
- 🔄 **CI/CD Pipeline** — GitHub Actions: lint → build → image → staging deploy
- 📖 **API Docs** — Swagger UI terintegrasi

**Stack:** `React 18` `Vite 7` `Tailwind CSS` `shadcn/ui` `Radix UI` `Node.js 20` `Express` `MySQL 8` `Redis` `Socket.IO` `Docker` `JWT` `Prometheus` `Winston` `GitHub Actions` `Swagger`
